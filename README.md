# High Speed Dadda Multiplier using Verilog HDL

This project implements a high-speed Dadda Multiplier using Verilog HDL for efficient binary multiplication in digital systems.

## Features
- High-speed multiplication
- Verilog HDL implementation
- Testbench verification
- Simulation and waveform analysis
- Digital VLSI design concept

## Tools Used
- Verilog HDL
- Cadence / Xilinx / ModelSim

## Project Files
- Verilog source code
- Testbench
- Simulation outputs
- Waveform screenshots
- Schematic outputs

## Applications
- Digital Signal Processing
- High-speed arithmetic circuits
- VLSI systems
- FPGA implementations
